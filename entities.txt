1. Categories - сущность, описывающая категории товаров
Первичный ключ - category_id
2. Cities - сущность, описывающая перечень городов
Первичны ключ - city_id
3. Countries - сущность, описывающая страны 
Первичный ключ - country_id
4. Customers - сущность, описывающая список производителей товаров
Первичный ключ - customer_id
5. Employees - сущность, описывает сотрудников 
Первичный ключ - employee_id
6. Products - сущность, описывающая товары магазина
Первичный ключ - product_id
7. Sales - сущность, содержащая информацию по продажам
Первичный ключ - sale_id
8. Shops - сущность, содержащая информацию о списке магазинов сети
Первичный ключ - shop_id

Первичные ключи определены по id сущности, так как id гарантирует уникальность каждой записи
